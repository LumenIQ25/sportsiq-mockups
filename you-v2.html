Design Mechanics · You tab
Header minimal — just SportsIQ logo + a settings gear (top-right). No chips here because the entire tab IS about "you."
Profile hero — 64px avatar (brand-gradient fill with first-letter monogram), nickname MarkN, title pill ⚡ Sharp (per MEMORY.md's five titles: Rookie → Sharp → WiseGuy → Legend → Elite — 2,450 IQ = Sharp). Phone number muted. Edit button changes nickname/avatar.
Stats grid 3×2 — 6 key metrics, each color-coded by type:
• IQ (amber) · Streak (orange) · Win Rate (green)
• Trivia Avg (7.8/10) · Picks Made (142) · Bars Played (3)
Recent list — last 3 picks: ✓/✗ outcome badges, plain-English title, multiplier + time ago, IQ delta (green/red). Tap any row to see the full reveal state for that pick.
Settings list (grouped):
• 🔔 Notifications · On ›
• 📍 Location · Bar Mode · On ›
• 📄 Terms & Privacy ›
• ↩ Sign out ›
• 🗑 Delete account › (red, destructive — matches production's existing flow)
Sign out behavior — clears session, routes back to phone entry. Returning user experience: phone + OTP only (skip nickname/terms since they already have them).
Delete account — production already has the flow (from bundle: "Enter the phone number associated with your SportsIQ account. We'll send a verification code to confirm your identity before deleting..."). Triggering this taps into existing backend.
Entire body is one scroll — no internal tabs. Keeps the surface predictable; users know everything about themselves is on this one page.
